---
description: How to run the application locally
---

1. Start XAMPP and run MySQL.
2. Create a database named `school_db` in phpMyAdmin.
3. In the `backend` directory, create a `.env` file based on `.env.example`.
4. Open a terminal in the `backend` directory.
// turbo
5. Run `npm install` to install dependencies.
// turbo
6. Run `npm start` to start the backend server.
7. Open `frontend/pages/admin/student.html` in your browser.
