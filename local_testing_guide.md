# دليل التشغيل والاختبار المحلي (Local Testing Guide)

لتشغيل المشروع على جهازك باستخدام **XAMPP** و **Node.js**، اتبع الخطوات التالية:

npm install
```

## 4. تشغيل السيرفر
- في نفس التيرمينال (مجلد `backend`)، قم بتشغيل السيرفر:
```bash
npm start
```
- ستظهر لك رسالة: `Server running on port 4000` و `Connecté à la DB MySQL (Pool) !`.

## 5. فتح الواجهة (Frontend)
- الآن يمكنك ببساطة فتح ملف `frontend/pages/admin/student.html` في متصفحك.
- ستجد الواجهة الجديدة تعمل وتتصل بالسيرفر المحلي.

---
> [!TIP]
> تأكد دائماً أن السيرفر (Node.js) وقاعدة البيانات (XAMPP) يعملان في نفس الوقت لتتمكن من إضافة أو حذف البيانات.
